
Backend-разработчик и инженер по автоматизации с практическим опытом создания отказоустойчивых сервисов для e-commerce и складской логистики. Специализируюсь на микросервисной архитектуре, ETL/DWH-пайплайнах и интеграциях с внешними системами (1С ERP, WMS, государственные API — Честный ЗНАК, GS1).
 
Инженерный бэкграунд (C/C++, embedded-системы, робототехника) позволяет видеть систему целиком — от аппаратных датчиков до бизнес-логики. Спроектировал и внедрил собственное IoT-устройство для измерения весогабаритных характеристик (точность 1 см / 5 г), систему управления возвратами на стыке трёх платформ и ETL-пайплайны, обрабатывающие 150+ ежедневных прайс-листов.
 
Работаю в команде или автономно, перевожу бизнес-требования в масштабируемый код. Открыт к удалённым позициям и сотрудничеству по B2B-контракту (ИП).
